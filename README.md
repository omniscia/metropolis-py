# Metropolis — Metropolis-Hastings MCMC sampler in Python

**Used in 0+ public GitHub repositories.** ![GitHub](https://img.shields.io/badge/in%20use-0%2B%20repos-blue)

An implementation of the Metropolis-Hastings algorithm for Markov-chain Monte Carlo simulation (Metropolis et al. 1953; Hastings 1970), with a travelling-salesman demo.

By **Vivake Gupta** (2001). This repository keeps the full lineage in version
control: the original release and a faithful Python 3 modernization
(`git log Metropolis.py` walks the evolution).

## Used by (a sample)
- A clean reference implementation of the algorithm; copied into MCMC examples and coursework.

## Usage
```python
from Metropolis import Metropolis
# see the module's __main__ for a worked TSP example
```
